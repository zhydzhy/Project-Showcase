Flagship Systems

The two projects that carry the clearest point of view.

Enterprise Web System / Manufacturing IT

01

Lean Activity Management Platform

Developed an enterprise-scale platform to digitize, standardize, and govern continuous-improvement cases across a manufacturing organization, supporting ~900 internal users as a centralized knowledge and analytics platform.

  • Designed a full-stack ASP.NET Core (.NET 10) and SQL Server system with secure cookie authentication and role-based access control.
  • Modeled organizational ownership through departments and positions instead of individual employee records for scalable governance.
  • Built Dapper-optimized search and analytics workflows that reduced retrieval from minutes to seconds.

CARLA / Multimodal Pipeline / Inference Evaluation

02

Traffic Detection System

GitHub Demo

Built a multimodal traffic hazard analysis pipeline combining RGB and LiDAR data for corner-case evaluation in autonomous driving scenarios, centered on synthetic data generation, cleaning, inference, validation, and benchmarking.

  • Built a CARLA synthetic dataset engine for rare events such as occlusions, adverse weather, near-miss collisions, and unusual agent behaviors.
  • Developed an RGB-LiDAR projection-based fusion module to produce aligned multimodal inputs for downstream comparison across models.
  • Used ontology-driven annotation and hazard scoring to validate outputs against structured judgment rather than intuition alone.
